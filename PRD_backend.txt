# 🧾 BACKEND PRD — WALLMIND (MVP, NO AI)

---

# 🎯 1. Goal

Build a backend that:

```text id="goal"
Accepts floor plan image → extracts geometry → applies rule-based material logic → returns 3D-ready JSON
```

---

# 🧱 2. Tech Stack

* **Runtime:** Node.js
* **Framework:** Express.js
* **File Upload:** Multer
* **Parser Engine:** Python (OpenCV) OR mock JSON
* **DB:** None (MVP)
* **Communication:** REST API

---

# 🏗️ 3. High-Level Architecture

```text id="arch"
Client (React + Three.js)
        ↓
Express API Server
        ↓
Parser Service (Python or Mock)
        ↓
Material Logic Service
        ↓
Response (JSON)
```

---

---

## 🔹 4.2 Controller Layer

**Responsibilities:**

## 🔹 4.3 Service Layer

### A. Parser Service

**Input:**

* Image file path

**Output:**

### B. Material Service

**Input:**

* Structured walls

**Output:**

# 🔄 5. Request → Response Flow

```text id="flow"
Client uploads image
        ↓
Multer saves file
        ↓
Controller triggers parser
        ↓
Parser returns wall coordinates
        ↓
Material service applies rules
        ↓
Response sent as JSON
```

# 📥 6. API Contract (Detailed)

---

## POST /api/parse

### Request

* Content-Type: multipart/form-data
* Field: `file`

---

### Response

```json id="response"
{
  "status": "success",
  "data": {
    "walls": [
      {
        "id": 1,
        "start": [0, 0],
        "end": [5, 0],
        "length": 5,
        "type": "outer"
      }
    ],
    "materials": [
      {
        "wallId": 1,
        "options": ["RCC", "Red Brick"]
      }
    ]
  }

```

---

### Error Response

```json id="error"
{
  "status": "error",
  "message": "Invalid file type"
}
```

# 🧠 7. Core Logic

---

## 🔹 7.1 Geometry Processing

### Wall Length

```js id="len"
Math.sqrt((x2-x1)**2 + (y2-y1)**2)
```

---

## 🔹 7.2 Wall Classification

**Simple MVP logic:**

* Long boundary walls → outer
* Short/internal → inner

```js id="classify"
if (length > threshold) → outer
else → inner
```

---

## 🔹 7.3 Material Rules

| Wall Type | Materials                |
| --------- | ------------------------ |
| Outer     | RCC, Red Brick           |
| Inner     | AAC Block, Fly Ash Brick |
